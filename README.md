## Installation 

1. Open Terminal

2. Follow the instructions for your operating system 

For Linux and MacOS
```bash 
  cd ~/.config/
  git clone https://github.com/yashguptaz/nvy nvim
```

For Windows
```bash  
  cd ~\AppData\Local\
  git clone https://github.com/yashguptaz/nvy nvim
```
    
## Features

- LSP
- Syntax Highlighting
- Auto Completion
- Toggle Terminal
- Snippets
- File and Project Search
- Explorer
- Git Plugins
- Zen Mode
- Configure with JSON
